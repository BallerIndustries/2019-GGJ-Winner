Requires NodeJS and NPM

# Play the game
- ```npm install```
- ```npm run start-dev```
- go to ```localhost:9876``` in your favourite browser

# Watch for changes
- ```npm run watch```

# Push to heroku
- Add heroku as a remote ```git add remote heroku https://git.heroku.com/musical-chairs.git```
- Push to heroku ```git push heroku master```
